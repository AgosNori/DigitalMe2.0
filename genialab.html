Un nuevo mundo te espera!

Es un obra esencial para educadores y estudiantes interesados en el fascinante mundo de la tecnología y la innovación. Escrito por la Prof. Gisela Fariaz, este libro ofrece una guía para implementar proyectos en el aula, utilizando metodologías ágiles donde se posiciona a los estudiantes como integrantes de diseño y desarrollo donde deben analizar y resolver desafíos.

Con 238 páginas de contenido enriquecedor, este libro se convierte en un recurso valioso para aquellos que buscan integrar la programación y la robótica en su enseñanza. La estructura clara y el índice detallado permiten una fácil navegación, facilitando la búsqueda de información específica y la planificación de actividades educativas.

Dificultad : Principiante
Duración : 2 semanas

Creación y manejo de repositorios

Aprende a gestionar el control de versiones con Git y Github desde cero. Este curso, conocerás los camandos básicos de Git, crearás tu primer archivo README y subirás tu proyecto a Github paso a paso. Ideal para quienes quieren comenzar a trabajar de manera colaborativa y profesional en el desarrollo de software.

Dificultad : Principiante
Duración : 2 semanas

Diseño y Desarrollo Web

Aprende a crear sitios web desde cero con HTML, CSS y JavaScript. En este curso, te enseñaremos los fundamentos del desarrrollo web, desde la estructura de una página hasta la creación de diseños atractivos y dinámicos. Ideal para principiantes que quieren dar sus primeros pasos en el mundo del desarrollo web.

Dificultad : Principiante
Duración : 2 semanas
